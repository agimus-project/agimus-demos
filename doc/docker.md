# Docker installation

Before you begin, ensure that Docker is installed on your system. Follow the official Docker installation guide if it is not already installed: [Install Docker Engine](https://docs.docker.com/engine/install/).

```bash
# clone agimus dev container repository
git clone --branch humble-devel https://gitlab.laas.fr/agimus-project/agimus_dev_container.git
# open the container and clone the `agimus-demos repo`
cd ~/ros2_ws/src
git clone --branch humble-devel https://github.com/agimus-project/agimus-demos.git
```
